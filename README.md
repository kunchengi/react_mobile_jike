# 极客园移动端

## 技术栈
* vite
* react
* react-router-dom
* typescript
* antd-mobile

## antd-mobile
* 安装
  ```bash
    npm i antd-mobile
  ```
* 使用
  ```ts
    import { Button } from 'antd-mobile';
    const App = () => {
      return (
        <Button>Hello World</Button>
      );
    }
  ```
